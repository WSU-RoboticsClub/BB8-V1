These programs are used to initially pair the HC-05 and the HC-06.

Upload each program to it's respective board and follow the instructions on: 
http://www.martyncurrey.com/connecting-2-arduinos-by-bluetooth-using-a-hc-05-and-a-hc-06-pair-bind-and-link/
to see complete instructions on how to set them to pair and then lock them in that configuration.

Paired_Comms is used to test the connection once they have been paired. It simply sends strings from the
Serial Monitor on one board to the Serial Monitor on the other
